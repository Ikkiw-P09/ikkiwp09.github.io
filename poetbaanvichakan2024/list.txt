Microsoft Word: โปรแกรมประมวลผลคำ ใช้สำหรับสร้างและแก้ไขเอกสาร เช่น จดหมาย รายงาน บทความ

Microsoft Excel: โปรแกรมตารางคำนวณ ใช้สำหรับจัดการข้อมูล ทำการคำนวณ และสร้างกราฟ

Microsoft PowerPoint: โปรแกรมนำเสนอ ใช้สำหรับสร้างสไลด์นำเสนองาน บทเรียน หรืออื่นๆ

Adobe Photoshop: โปรแกรมแก้ไขภาพแบบ raster ใช้สำหรับแต่งรูป ตัดต่อภาพ และสร้างงานกราฟิก

Adobe Illustrator: โปรแกรมสร้างภาพแบบเวกเตอร์ ใช้สำหรับออกแบบโลโก้ ภาพประกอบ และงานกราฟิกอื่นๆ

Adobe Premiere Pro: โปรแกรมตัดต่อวิดีโอระดับมืออาชีพ พัฒนาโดย Adobe

Audacity: โปรแกรมบันทึกและตัดต่อเสียง

Blender: โปรแกรมสร้างแบบจำลอง 3 มิติ ใช้สำหรับสร้างโมเดล 3D แอนิเมชัน และอื่นๆ

Visual Studio Code (VS Code): โปรแกรมแก้ไขโค้ด ใช้สำหรับเขียนโปรแกรม

FileZilla: โปรแกรมสำหรับเชื่อมต่อและถ่ายโอนไฟล์ผ่าน FTP

WinRAR: โปรแกรมบีบอัดและแตกไฟล์

VLC Media Player: โปรแกรมเล่นไฟล์มัลติมีเดีย รองรับไฟล์ได้หลากหลายรูปแบบ

Discord: แพลตฟอร์มสื่อสารสำหรับเล่นเกมและกลุ่มต่างๆ

Steam: แพลตฟอร์มจำหน่ายเกมดิจิตอล

OBS Studio: โปรแกรมบันทึกหน้าจอและสตรีมมิ่ง

Microsoft OneNote: โปรแกรมจดบันทึก

Google Chrome: เว็บเบราว์เซอร์

Mozilla Firefox: เว็บเบราว์เซอร์

Spotify: แพลตฟอร์มสตรีมเพลง

Netflix: แพลตฟอร์มสตรีมภาพยนตร์และซีรีส์

Notepad++: โปรแกรมแก้ไขข้อความ

7-Zip: โปรแกรมบีบอัดและแตกไฟล์

GIMP: โปรแกรมแก้ไขภาพแบบ raster พัฒนาโดย GNU

Inkscape: โปรแกรมสร้างภาพแบบเวกเตอร์

Krita: โปรแกรมวาดภาพดิจิทัล

DaVinci Resolve: โปรแกรมตัดต่อวิดีโอระดับมืออาชีพ

Unity: เกมเอนจิน ใช้สำหรับพัฒนาเกม

Unreal Engine: เกมเอนจิน ใช้สำหรับพัฒนาเกม พัฒนาโดย Epic Games

GitHub Desktop: โปรแกรมสำหรับจัดการ Git repository
