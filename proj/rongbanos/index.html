เข้ารหัสการสื่อสาร TLS 1.3

Zero-Downtime Updates

Immutable Logging

คำถามที่พบบ่อย

การย้ายจาก Windows Server มาใช้ Rongban OS ยุ่งยากไหม?

ทีมงานของเรามีบริการช่วยเหลือในการย้ายระบบ (Migration) เพื่อให้การเปลี่ยนแปลงเป็นไปอย่างราบรื่นและส่งผลกระทบต่อการทำงานน้อยที่สุด โดยเราจะวางแผนร่วมกับทีม IT ของโรงพยาบาลอย่างใกล้ชิด

ซอฟต์แวร์จัดการโรงพยาบาล (HIS) ที่ใช้อยู่จะทำงานบน OS นี้ได้หรือไม่?

Rongban OS ถูกออกแบบให้รองรับการทำงานร่วมกับซอฟต์แวร์ HIS ส่วนใหญ่ผ่านเทคโนโลยี Virtualization หรือ Containerization กรุณาติดต่อทีมงานเพื่อตรวจสอบความเข้ากันได้ของซอฟต์แวร์ที่ท่านใช้งานอยู่

ต้องใช้ฮาร์ดแวร์สเปคแบบไหน?

เรามีฮาร์ดแวร์เซิร์ฟเวอร์รุ่น RPC-01 Pro ที่ปรับแต่งมาเพื่อประสิทธิภาพสูงสุดของ Rongban OS หรือหากท่านมีเซิร์ฟเวอร์อยู่แล้ว ทีมงานของเราสามารถให้คำแนะนำสเปคขั้นต่ำและช่วยเหลือในการติดตั้งได้
